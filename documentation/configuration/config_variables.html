LMS System Configuration Variables

This page documents all the configurable variables in the LMS system. These variables can be set through environment variables, the .env file, or directly in the configuration files.

Port Configuration

Variable Name Default Value Description Where to Configure
LMS_API_PORT 8000 The port on which the backend API server runs .env, environment variable
LMS_API_HOST 0.0.0.0 The host interface on which the backend API server listens .env, environment variable
LMS_FRONTEND_PORT 3000 The port on which the frontend Next.js server runs .env, environment variable
LMS_FRONTEND_HOST localhost The host interface on which the frontend server listens .env, environment variable
LMS_DB_PORT 5432 The port on which PostgreSQL database server runs .env, environment variable
LMS_DB_HOST localhost The host on which PostgreSQL database server runs .env, environment variable

Database Configuration

Variable Name Default Value Description Where to Configure
LMS_DB_NAME lms_db PostgreSQL database name .env, environment variable
LMS_DB_USER postgres PostgreSQL database user .env, environment variable
LMS_DB_PASSWORD No default PostgreSQL database password *Required .env, environment variable
LMS_DB_POOL_SIZE 5 Maximum number of database connections in pool .env, environment variable
LMS_DB_TIMEOUT 30 Database connection timeout in seconds .env, environment variable

CORS Configuration

Variable Name Default Value Description Where to Configure
LMS_CORS_ORIGINS http://localhost:3000 Comma-separated list of allowed origins for CORS .env, environment variable
LMS_CORS_METHODS GET,POST,PUT,DELETE Allowed HTTP methods for CORS requests .env, environment variable
LMS_CORS_HEADERS Content-Type,Authorization Allowed HTTP headers for CORS requests .env, environment variable

Authentication Configuration

Variable Name Default Value Description Where to Configure
LMS_SECRET_KEY No default Secret key for JWT token generation *Required .env, environment variable
LMS_JWT_ALGORITHM HS256 Algorithm used for JWT token encoding .env, environment variable
LMS_ACCESS_TOKEN_EXPIRE_MINUTES 60 JWT access token expiration time in minutes .env, environment variable
LMS_REFRESH_TOKEN_EXPIRE_DAYS 7 JWT refresh token expiration time in days .env, environment variable

Path Configuration

Variable Name Default Value Description Where to Configure
LMS_STATIC_DIR /app/static Directory for static files .env, environment variable
LMS_MEDIA_DIR /app/media Directory for user-uploaded media files .env, environment variable
LMS_LOG_DIR /app/logs Directory for application logs .env, environment variable
LMS_TEMP_DIR /app/tmp Directory for temporary files .env, environment variable

Logging Configuration

Variable Name Default Value Description Where to Configure
LMS_LOG_LEVEL INFO Log level (DEBUG, INFO, WARNING, ERROR, CRITICAL) .env, environment variable
LMS_LOG_FORMAT json Log format (json, text) .env, environment variable
LMS_LOG_RETENTION_DAYS 30 Number of days to retain log files .env, environment variable
LMS_ACCESS_LOG true Whether to log API access (true, false) .env, environment variable

Frontend Configuration

Variable Name Default Value Description Where to Configure
NEXT_PUBLIC_API_URL http://localhost:8000 URL of the backend API as seen from the browser .env, environment variable
NEXT_PUBLIC_SITE_NAME LMS System Name of the site displayed in UI .env, environment variable
NEXT_PUBLIC_SUPPORT_EMAIL support@example.com Support email displayed in UI .env, environment variable

Exercise Engine Configuration

Variable Name Default Value Description Where to Configure
LMS_MAX_SUBMISSION_SIZE 10485760 Maximum submission size in bytes (10MB) .env, environment variable
LMS_EVALUATION_TIMEOUT 30 Maximum time in seconds for exercise evaluation .env, environment variable
LMS_MAX_ATTEMPTS 3 Maximum number of attempts for exercises .env, environment variable
